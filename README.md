# 📚 Book Website

## ✨ Overview
This is a front-end book website project designed as a visual and interactive platform for browsing and displaying books.

The project explores layout design, content structuring, and UI development for a digital reading/book showcase experience. It acts as a foundation for a future expansion into a fully functional **book tracking and scanning application**, where users can build a personalised digital bookshelf.

---

## 🎯 Purpose
This project was created to:
- Practice front-end web development (HTML, CSS, JavaScript)
- Build structured UI layouts for content-heavy applications
- Explore early-stage design for a book tracking / library system
- Develop ideas for a future mobile-first reading ecosystem

---

## 💡 Features
- 📖 Book display layout / catalogue-style interface  
- 🧭 Simple navigation between pages or sections  
- 🎨 Styled UI for book presentation  
- 🖼️ Visual focus on book covers and metadata  
- 🧱 Static structure designed for future backend integration  

---

## 🚀 Future Vision
This project is intended to evolve into a larger system, including:

### 📱 Mobile App Expansion
- Scan books using barcode/ISBN
- Automatically add books to a personal library
- Sync reading data across devices

### 📚 Smart Bookshelf System
- Visual bookshelf interface
- Colour-coded reading status:
  - Full colour → read books  
  - Greyscale → unread books  

### 📊 Reading Tracker Features
- Track reading progress
- Save reading history
- Monitor reading habits

### 🌐 Social Features (future idea)
- Share reading lists
- Compare reading progress with friends
- Recommend books based on activity
